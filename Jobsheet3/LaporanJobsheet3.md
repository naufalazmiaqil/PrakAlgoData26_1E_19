2. Membuat array yang bisa menampung 3 object Mahasiswa
3. tidak, karena di dalam class Mahasiswa tidak ditulis konstruktor secara eksplisit,
Java otomatis menyediakan konstruktor default (tanpa parameter).
4. Membuat object Mahasiswa baru dan menyimpannya di index ke-0 array.

    - nim
    - nama

    - ipk
