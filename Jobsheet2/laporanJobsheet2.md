# 2.3 Percobaan 3 – Konstruktor

## Penjelasan
Pada percobaan ini ditambahkan konstruktor pada class Mahasiswa.  
Konstruktor digunakan untuk menginisialisasi nilai atribut saat object dibuat.

Terdapat dua jenis konstruktor:
- Konstruktor default (tanpa parameter)
- Konstruktor berparameter (dengan parameter)

Dengan konstruktor berparameter, data dapat langsung diisi saat pembuatan object.
